## 앱 정보

**Pure Noise**는 편안한 수면을 위한 백색소음 앱입니다.

### 주요 기능
- 다양한 자연음 제공 (비 소리, 바다 소리, 심장 소리 등)
- 타이머 기능
- 백그라운드 재생
- 사용자 친화적 인터페이스
